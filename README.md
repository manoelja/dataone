# DATAONE

### Central Inteligente de Tratamento de Dados

O DATAONE é uma plataforma web que transforma dados brutos em informações úteis — sem complicação. Faça upload de planilhas CSV ou Excel, limpe, formate, analise e exporte tudo com poucos cliques.

---

## O que o DATAONE faz?

**Upload Inteligente**
Carregue arquivos CSV ou Excel. O sistema detecta automaticamente separadores, encodings e abas — você só precisa arrastar o arquivo.

**Limpeza de Dados**
Exclua colunas desnecessárias, insira novas colunas e organize sua tabela com facilidade.

**Formatação de Texto**
Ajuste caixa (maiúsculo/minúsculo), gere IDs sequenciais, aplique máscaras de CPF e CNPJ, ou gere documentos fictícios para testes.

**Configurações Visuais**
Defina formatos de moeda (R$, US$, Euro) e datas (DD/MM/YYYY, YYYY-MM-DD) para suas colunas.

**Análise Automática**
Gráficos interativos com Plotly, detecção de outliers, correlações, distribuições e insights inteligentes que aparecem automaticamente.

**Central de Fórmulas Excel**
Consulte rapidamente as principais fórmulas do Excel com sintaxe, exemplos e categorias.

**Exportação**
Baixe seus dados tratados em Excel já formatado com as configurações de moeda e data que você definiu.

---

## Como usar

1. Acesse a plataforma
2. Faça upload de um arquivo CSV ou Excel
3. Use as abas para limpar, formatar e analisar seus dados
4. Clique em "Baixar Excel" para exportar o resultado
